# teocuitlatl_app
📊 Real-Time Asset Price Tracker
A lightweight web application built with plain HTML, CSS, and JavaScript that displays real-time prices for various assets such as gold. The app helps users monitor market trends and identify favorable moments to make additional purchases.

It fetches live data from external APIs and presents it through a clean, responsive interface. Users can easily track price fluctuations and make informed decisions based on current market conditions.

🚀 Features
Real-time price tracking for assets (e.g., gold)
Simple and responsive UI
Lightweight (no frameworks required)
Dynamic data fetching using JavaScript
Visual cues to help identify good buying opportunities
🛠️ Tech Stack
HTML5
CSS3
Vanilla JavaScript
📌 Use Case

Ideal for users who want a quick and simple way to monitor asset prices and decide when to invest more based on market movements.
